A curated collection of binary and multi-class classification projects leveraging Logistic Regression to solve real-world decision-making problems. Features robust pipelines for handling imbalanced datasets, feature scaling, and rigorous evaluation using ROC-AUC and Precision-Recall metrics
Titanic Survival --  An end-to-end machine learning project that uses Logistic Regression to predict Titanic passenger survival based on demographic and travel-related features. The project covers data cleaning, visualization, feature engineering, model evaluation, and prediction.
Breast Tumor -- A data science project that applies Logistic Regression to classify breast tumors based on diagnostic features. The project covers the complete machine learning pipeline, including preprocessing, EDA, model training, evaluation, and prediction.
